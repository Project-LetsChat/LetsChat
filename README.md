# LetsChat PREVIOUS Branch
The LetsChat previous branch includes older versions of LetsChat.

## Directories
Directories contain older versions, the list below will list what contains what.

### Alpha(s)
There are 3 alpha builds of LetsChat, which are saved from the development phase before the release.

### BETA
This directory contains the 3 beta builds of LetsChat.

### Pre-Release
The pre-release is the last version before the release version.

## Original
This directory contains the latest version of the original LetsChat, read the README file in the <code>Original</code> directory for more detailed information.
